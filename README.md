# MOSSDeF OpenWrt packages sources

This repo contains source code for the OpenWrt packages maintained under MOSSDeF (Melmac Open Source Software Development Foundation). For more information check out [documentation](https://docs.mossdef.org) or [browse code at GitHub](https://github.com/mossdef-org/source.mossdef.org).

Individual packages and are available in their own repositories:

- [adblock-fast](https://github.com/mossdef-org/adblock-fast/)
- [luci-app-adblock-fast](https://github.com/mossdef-org/luci-app-adblock-fast/)
- [fakeinternet](https://github.com/mossdef-org/fakeinternet/)
- [luci-app-fakeinternet](https://github.com/mossdef-org/luci-app-fakeinternet/)
- [https-dns-proxy](https://github.com/mossdef-org/https-dns-proxy/)
- [luci-app-https-dns-proxy](https://github.com/mossdef-org/luci-app-https-dns-proxy/)
- [luci-app-advanced-reboot](https://github.com/mossdef-org/luci-app-advanced-reboot/)
- [netclient](https://github.com/mossdef-org/netclient/)
- [pbr](https://github.com/mossdef-org/pbr/)
- [luci-app-pbr](https://github.com/mossdef-org/luci-app-pbr/)
- [slider-support](https://github.com/mossdef-org/slider-support/)
- [wireshark-helper](https://github.com/mossdef-org/wireshark-helper/)
- [luci-app-wireshark-helper](https://github.com/mossdef-org/luci-app-wireshark-helper/)
- [wlanblinker](https://github.com/mossdef-org/wlanblinker/)
- [luci-app-wlanblinker](https://github.com/mossdef-org/luci-app-wlanblinker/)
